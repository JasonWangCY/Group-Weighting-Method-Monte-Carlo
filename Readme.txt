Written in C and compiled in CodeBlocks.

Determining the accuracy of different weighting methods of group projects
using a Monte Carlo approach.

Attached in this file are the code, the report, the graphs and the following text files:
1. Table - an example of the peer assessments generated by the Monte Carlo code
2. Error vs contrib - comparing errors of individual students' gradings with their real contributions
3. Gradings - comparing the average errors of a group's gradings with average self-assessment inflation
